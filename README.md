## 🚀 Getting Started

### **Quick Start**

1. Visit: [SocialMind AI](https://royalpuskar.github.io/Social-Mind-AI/)
2. Paste your message/email/post
3. Enter your Anthropic API key
4. Click "Analyze Now"
5. Get instant AI-powered safety verdict

### **Enterprise Deployment**

```bash
# Install via npm
npm install socialmind-ai

# Import in your project
import SocialMindAI from 'socialmind-ai'

# Use the API
const analysis = await SocialMindAI.analyze(message, {
  language: 'en',
  models: ['bert', 'gpt', 'transformers'],
  features: ['scam', 'phishing', 'deepfake', 'sentiment']
})
```
